# Tabbed File Manager

## デプロイルール
コード変更後は必ず以下の順序で実行すること：
1. **git add → git commit → git push** （GitHubに反映）
2. **clasp push** （GASプロジェクトに反映）

両方を必ずセットで行う。片方だけで終わらせない。
